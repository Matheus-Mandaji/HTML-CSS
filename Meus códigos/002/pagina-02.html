HTML

O HTML é uma linguagem de marcação utlizada para o desenvolvimento de sites. Esta linguagem surgiu junto com o HTTP, ambos possibilitaram a internet ser popularizada.

História

O HTML fo criado em 1991, por Tim Berners-Lee, no CERN na Suiça. Inicialmente o HTML foi projetado para interligar instituições de pesquisas próximas, e compartilhar documentos com facilidade. Em 1992, foi liberada a biblioteca de desenvolvimento WWW, uma rede de alcance mundial, que junto com o HTML proporcionou o uso em escala mundial da WEB.

Como funciona

O HTML é uma linguagem de marcação. Estas linguagens são construídas de códigos que delimitam conteúdos específicos, segundo uma sintaxe própia. O HTML tem códigos para criar páginas na web. Estes códigos que defienem o tipo de letra, qual a cor, espaçamento, e vários outros aspectos do site No início era muito complicado aprender HTML, pois eram muitos comandos para fazer algo imples. A cada nova versão, o HTML fica mais fácil de utilizar, e adiquire mais funções. Atualmente qualquer pessoa pode acessar a internet e aprender a consruir um site básico em questão de horas, seguindo os passos de tutoriais aprendendo as funções de cada código.

O HTML foi sprimeira linguagem de nível mundial, porem não é a única. Existem muitas outras linguagens destinadas a criação de páginas da Web, porém o HTML ainda prevalece. Atualmente á é possível integrar várias linguagens na mesma página da web, sendo possivel usar duas ou mais linguagens no mesmo site.

Para criar e editar códigos em HTML é necessário qualquer editor de texto comum, coo bloco de notas. Para testar os códigos. basta salvar o aqruivo em formato .HTML e executar. Para o teste é necessário ter m navegador configurado como padrão. Não é necessária internet, pois o arquivo com os códigos esta na maquina onde esta sendo executado.

Alguns códigos em HTML e suas funções:

º <title> define o título da página;

º <Script> Permite adicionar funções em páginas com scrpit, podendo assim adicionar códigos em JavaScript (este código permite que alguns sites HTML tenham joguinhos ou animações, verificações de foemulários antes do envio para o servidos, entre outras funcionalidades);

º <stylr> Define formatação em CSS;

A maioria dos códigos em HTML precisam ter um código de inicialização e um de fialização - são chamadas Tag's (tag de abertura e tag de fechamente=o). Por exemplo, para definir o título da página, é necessário escrever:

<title> Título da minha página </title>

O </title> serve para "avisar" que ali acaba o título, caso não seja posto, todos os códigos definidos depois serão consderados partre do título.

Um código HTML para uma página onde apareça "infoescola" escrito em vermelho, em um fundo preto:

        
            <HTML> 
            <Head>
            <Title>Título</Title>
            </Head><body bgcolor= "black" text="red">
            Infoscola 
            </body>
            </HTML>

Basta copiar o código para um editor de texto, salvar em .HTML e abrir no navegador.
